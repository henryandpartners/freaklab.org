# FREAK Lab — Static Site

**Futuristic Research in Enigmatic + Aesthetic Knowledges**

Static HTML version of [freaklab.org](https://freaklab.org) — migrated from WordPress.

### Structure
- `/` — Home page
- `/research/` — Research groups
- `/projects/` — All projects with filter
- `/people/` — Team members
- `/events/` — Events timeline
- `/updates/` — News and updates
- `/join-us/` — Join the community

### Deploy
This is a flat static site. Deploy to any static host (GitHub Pages, Vercel, Netlify).
